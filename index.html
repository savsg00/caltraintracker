⚠️ CORS Note — Running on Replit? The 511 API blocks direct browser requests. You need a tiny proxy. Add this to your Replit server.js:

app.get('/proxy', async (req,res) => { const url = req.query.url; const r = await fetch(url); const data = await r.json(); res.json(data); });

Then set Proxy URL to your Replit URL + /proxy. Or use the demo mode below to explore the UI with sample data.
